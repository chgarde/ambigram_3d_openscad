# Ambigram 3D generator for OpenScad

This is a customizable 3D ambigram generator for OpenScad.

## Instructions
* Download and install openscad
* Open ambigram.scad
* Customize texts
* Hit F6
* File > Export as STL or AMF
* Slice it with your preferred slicer
* Print it and enjoy :)
